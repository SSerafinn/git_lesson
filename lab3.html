Checking the status of the repository

1

Goals

Learn how to check the repository’s status.

01

Check the status of the repository

Use the git status command to check the current state of the repository.

Run
git status

You will see:

Result 
$ git status
On branch main
nothing to commit, working tree clean

If you see On branch master instead of On branch main, you may have an older Git that didn’t apply the default branch setting. Rename the branch to main with:

Run
git branch -m master main

The command checks status and reports there’s nothing to commit, which means the repository stores the current state of the working directory and there are no changes to record. We’ll use git status frequently to monitor both the working directory and the repository.
